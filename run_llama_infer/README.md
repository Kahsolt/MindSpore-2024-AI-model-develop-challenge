# Run LLaMA infer

    为了解决广大人民群众玩不了大模型推理的根本问题，本教程研究如何在 Windows 平台上使用 CPU/GPU 设备运行 mindformers 所提供的 LLaMA2-7b 和 LLaMA3-8b 模型 🎉🎉

----

### Setup Mindformers

- `git clone https://gitee.com/kahsolt/mindformers`
  - ⚠ 需要使用我们魔改之后的 mindformers，不然有些 OS-specified API 在 Windows 上跑会报错
- 下载权重
  - run `run_llama_infer\checkpoint_download\llama2\download.cmd`
  - run `run_llama_infer\checkpoint_download\llama3\download.cmd`
  - (*) 如果你已经提前下载到了别的地方，不用重复下载或者复制文件，我们提供命令行参数来动态指定加载文件位置


### Run tokenizer

- run `python run_tokenizer.py`
  - `-V <2|3>` 指定 llama 版本
  - `-F <vocab_file>` 自定义 `tokenizer.model` 文件路径
  - `--fast` 是否使用 fast 版本 (仅支持 llama2)


### Run model

⚠ This runs but is extremely slow: 2~3 min per token on my i7-11700K@3.6GHz (8-core/16-thread) for LLaMA2-7b  
⚠ LLaMA3-8b still CANNOt run, there's some ckpt load error :(  

- run `python run_model.py`
  - `-V <2|3>` 指定 llama 版本
  - `-C <config_file>` 指定 yaml 配置文件路径
  - `-F <vocab_file>` 自定义 `tokenizer.model` 文件路径
  - `-M <ckpt_file>` 自定义 `llama_*.ckpt` 文件路径
